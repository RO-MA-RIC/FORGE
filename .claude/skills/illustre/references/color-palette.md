# Palette de Couleurs L'Accélérateur IA — Excalidraw

Palette de marque ancrée sur le bleu L'Accélérateur IA (`#0061FF`). C'est la **source unique** de la palette du skill `illustre` — à lire avant de colorer un diagramme.

## Couleurs des Shapes (Sémantique)

| Usage | Fill | Stroke | Quand utiliser |
|-------|------|--------|----------------|
| Primary/Action | `#0061FF` | `#0047CC` | Éléments principaux, actions, CTA (couleur de marque) |
| Secondary/Support | `#06B6D4` | `#0E7490` | Éléments de support, secondaires (cyan analogue) |
| Accent/Highlight | `#FF8A00` | `#CC6E00` | Points d'attention, highlights (orange complémentaire) |
| Neutre/Container | `#F1F5F9` | `#334155` | Containers génériques, fonds |
| Succès/Résultat | `#A7F3D0` | `#047857` | Résultats positifs, fin de processus |
| Attention/Warning | `#FEE2E2` | `#DC2626` | Alertes, erreurs, points critiques |
| Donnée/Input | `#CCE0FF` | `#0052D6` | Entrées, données, sources (teinte de marque) |
| Décision | `#FEF3C7` | `#B45309` | Points de décision, conditions |
| AI/LLM | `#C4B5FD` | `#7C3AED` | Éléments liés à l'IA (violet franc) |
| Inactive | `#E2E8F0` | `#94A3B8` (dashed) | Éléments désactivés |

## Couleurs du Texte (Hiérarchie)

| Niveau | Couleur | Usage |
|--------|---------|-------|
| Titre | `#0F172A` | Titres principaux, headers |
| Sous-titre | `#0061FF` | Sous-titres, labels importants (couleur de marque) |
| Corps | `#334155` | Descriptions, annotations |
| Détail | `#64748B` | Notes secondaires, metadata |
| Sur fond clair | `#374151` | Texte dans shapes claires |
| Sur fond foncé | `#FFFFFF` | Texte dans shapes foncées (Primary bleu, strokes foncés) |

## Flèches et Lignes

| Élément | Couleur | Usage |
|---------|---------|-------|
| Flèche standard | Stroke de l'élément source | Connexions |
| Ligne structurelle | `#334155` | Séparateurs, timelines |
| Ligne secondaire | `#64748B` | Connexions secondaires |
| Marqueur dot | `#0061FF` | Points de repère |

## Canvas

- **Fond** : `#FFFFFF`
- **Grille** : non visible (clean)

## Lisibilité

- Texte **blanc** (`#FFFFFF`) sur les fills foncés et saturés : Primary (`#0061FF`), strokes foncés.
- Texte **foncé** (`#0F172A` / `#374151`) sur les fills clairs : Accent orange, Décision, et tous les pastels.
